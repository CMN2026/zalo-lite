# Chat Application ZaloLite

## Tech Stack
- Frontend: Next.js, React Native
- Backend: Express.js
- Database: MySQL, DynamoDB
- Architecture: Microservices

## Features
- User profile & friend management
- 1-1 & group chat
- Chatbot support
- Real-time messaging

## How to run
- web: ..frontend/web -> npm run dev
- mobile: ..frontend/mobile -> npm start
- backend: ..backend/api-gateway -> npm run dev
